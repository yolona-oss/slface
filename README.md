## Installation

```BASH
cd /path/to/slface/build
cmake .. -DCURL_CA_BUNDLE="/path/to/cacert.pem"
make
```

<a name="dependenses"></a>
### Dependenses
+ Qt6
+ SQLite
+ RapidJSON (downloading by cmake)
+ cpr (downloading by cmake)
